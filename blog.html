1. What is DOM? What is the purpose of DOM?

DOM is Document Object Model. DOM is a programmimg interface for HTML document.
When webpage load in a browser, HTML document is converted into DOM tree. By using DOM, Javascript can easily manipulates the HTML elements.
DOM allows programs and script to dynamically access and update the content, structure, and style of a document.

2. How can you select element from DOM?

There are five ways to select element from DOM.

  1. getElementsByTagName(): Using Tag Name DOM select element
  2. getElementById(): Using the id of the element DOM select element
  3. getElementsByClassName(): Using the class name of the element DOM select element
  4. querySelectorAll(): Using the CSS selectors it return the Nodelist
  5. querySelector(): it returns the first element that matches the specified selector.

3. What is event bubble?

Event bubble is a process of event propagation in HTML DOM. When event is added to an element inside another element, both element are under that event handler. It is a process that starts with the element that triggered the event and then bubbles up to the containing elements in the hierarchy.

4. What is callback function and its uses?

A callback function is a function passed into another function as an arguments. A callback function is to be executed after another function has complete its execution. It is the way to passes a function to another function as an arguments.
